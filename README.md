## 📌 Project Overview

This system simulates a real-world library management workflow. It allows librarians to manage books, handle member registrations, issue and return books, and track availability — all via a clean, menu-driven console interface built with Java OOP principles.

---

## ✨ Features

- ➕ **Add / Remove Books** — manage the library catalogue
- 👤 **Member Registration** — add and manage library members
- 📖 **Issue & Return Books** — track book lending with due dates
- 🔍 **Search Books** — search by title, author, or ISBN
- 📋 **View All Records** — list all books and members
- ⚠️ **Availability Check** — real-time book availability status

---

## 🧱 OOP Concepts Applied

| Concept | Where Used |
|---|---|
| **Classes & Objects** | Book, Member, Library classes |
| **Encapsulation** | Private fields with getters/setters |
| **Inheritance** | Member types (e.g., Student, Faculty) |
| **Polymorphism** | Overridden display/search methods |
| **Abstraction** | Interface for library operations |

## 🚀 How to Run

```bash
# 1. Clone the repository
git clone https://github.com/nandkishor-ux/library-management-system.git
cd library-management-system

# 2. Compile
javac Main.java

# 3. Run
java Main
```

**Requirements:** Java JDK 8 or above
